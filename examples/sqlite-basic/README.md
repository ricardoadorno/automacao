# SQLite basic

## Como preparar

- `node examples/sqlite-basic/init-db.js`

## Como rodar

- `npm start -- --plan examples/sqlite-basic/plan.json --out runs`

## O que esperar

- `query.sql`, `result.csv`, `evidence.html` em `runs/<runId>/steps/01_query-users/`.
## Nota

- Este exemplo usa query inline no plan.
